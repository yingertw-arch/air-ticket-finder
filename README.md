# 機票一鍵搜尋啟動台

輸入一次出發地／目的地／日期，一鍵打開所有可信任訂票平台「已帶入你搜尋條件」的分頁，
不用一個一個網站重複輸入。看到喜歡的票後，**自行在該平台刷卡付款**。

> 本工具只把你帶到各平台的搜尋頁，**不代刷卡、不代下單、不收集你的資料**。

---

## 怎麼用

1. 雙擊 `index.html`（或用瀏覽器打開），即可本機使用。
2. 選行程模式：**單程 / 來回 / 多點進出**。
3. 輸入出發地、目的地（打「台北 / Taipei / TPE」都會自動對應代碼）、日期、人數、艙等。
4. 按「產生各平台搜尋連結」。
5. 點任一平台卡片，在新分頁開啟搜尋結果。或按「全部開啟」（需允許瀏覽器彈窗）。

## 降價提醒

本工具**不自建價格監看**，而是用平台內建的「價格追蹤」（最可靠、零維護）：

- **Google Flights** 與 **Skyscanner** 都能自動每天監看票價、降價時 email 你。
- 在結果頁的「🔔 降價提醒」區點連結 → 到該平台搜尋頁 → 登入帳號 → 按該平台的
  「追蹤價格 / Price alerts」按鈕即可。

（若未來想自建「收藏航線、每天中午自動查價並通知」，見計畫檔 Phase 2。）

---

## 如何新增 / 修改平台

- 機場代碼資料：編輯 `airports.js` 的 `AIRPORTS` 陣列。
- 平台網址規則：編輯 `app.js` 的 `buildLinks()`，每個平台一段，依模式（oneway/round/multi）組網址。

---

## 部署成線上網站（GitHub Pages）

1. 把本資料夾推到 GitHub repo（可用 `/push-github`）。
2. repo → Settings → Pages → Source 選 `main` 分支、根目錄。
3. 取得網址後手機/電腦皆可開。

> 純前端、無密鑰，可安全公開。
